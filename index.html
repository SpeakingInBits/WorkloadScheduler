How to Use

  1. Add Instructors - Enter names in the Instructors panel on the left
  2. Add Courses - Enter course details and optionally assign to an instructor
  3. Add Classrooms - Create rooms with room numbers
  4. Add Time Slots - Set up schedule times for each day of the week
  5. Schedule Courses - Drag courses from the sidebar onto schedule slots, then select modality
  6. Move Courses - Drag scheduled courses between timeslots to reorganize
  7. Edit Courses/Instructors - Double-click any course or instructor (in sidebar or schedule) to edit details
  8. Export/Import - Save your data as JSON or load previous schedules

Tips

  • Use the "Copy to All" button on Monday to quickly set up the same timeslots for the entire week
  • Courses can be scheduled multiple times (e.g., a course meeting MWF)
  • Double-click scheduled courses to quickly edit course information and modality
  • Double-click courses or instructors in the left sidebar to edit their basic information
  • Drag courses between timeslots to move them (preserves modality)
  • Use the "Arranged" column for courses that don't have fixed meeting times
  • Toggle classroom visibility to focus on specific rooms using the eye icon
  • Export your data regularly as a backup - it's saved as a JSON file you can import later
  • Courses are color-coded: green = scheduled, red = unscheduled
  • In-person scheduling conflicts are highlighted with a red background
